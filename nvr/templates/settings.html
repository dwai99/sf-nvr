Storage Optimization Tips
Motion-only mode can reduce storage by 70-90%. Use it for indoor cameras like storage rooms and stairways.
480p resolution is sufficient for indoor areas with good lighting. Use 720p for outdoor/entrance cameras.
Note: Resolution changes require a server restart to take effect. Recording mode changes apply immediately.
